# Code Agent for Boot.dev Backend Path Course

This is the code agent for the Boot.dev Backend Path course. It is a chatbot that can help you answer questions about your codebase and fix small bugs and issues. 

## Installation

    uv venv
    source .venv/bin/activate
    uv sync 

## Usage

    uv run main.py "in the calculator app, Fix the bug: 3 + 7 * 2 shouldn't be 20."
